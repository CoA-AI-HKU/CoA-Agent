# 記憶診所 - 賽馬會耆智園 | 一站式腦退化症服務中心

[參觀申請](https://www.jccpa.org.hk/services/environment-and-facilities/#application)

## 及早診斷 延緩退化速度

###### 若你懷疑自己或家人有腦退化的徵兆，只要及早由專業醫護作出診斷，便能盡快釋除心中的疑慮。「記憶診所」由註冊醫生為懷疑有腦退化症的人士提供記憶評估、照顧程度評估及臨床診斷，可讓跨專業醫護團隊及社工盡早介入，有助延緩確診腦退化症人士的退化速度。

## 服務對象​

懷疑有腦退化症或出現認知障礙人士
（本服務只限持有香港身份證之人士）
