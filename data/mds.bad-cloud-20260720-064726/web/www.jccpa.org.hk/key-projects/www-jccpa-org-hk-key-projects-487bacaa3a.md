# 焦點項目 - 賽馬會耆智園 | 一站式腦退化症服務中心

## 推動社區安老的先導者

###### 除了核心服務外，耆智園也會不時推行多元化的創新項目，旨在緊貼腦退化症人士在社會上的需求，彌補現時公共服務的缺口。過去10年，耆智園已完成超過10個焦點項目 ，不僅為腦退化症帶來全新、正面的名字，也動員全城推動多個先導計劃，全方位協助及支援腦退化症人士，讓他們得以在社區好好地老。

| 2019-2022 | 賽馬會「與耆同行」腦退化症支援計劃 | 承蒙香港賽馬會慈善信託基金捐助，賽馬會耆智園聯同基督教香港信義會社會服務部、基督教靈實協會以及九龍樂善堂，推行賽馬會「與耆同行」腦退化症支援計劃。這項創新的試驗計劃為期3年，主要支援懷疑有輕度認知障礙人士、早期腦退化症人士及他們的照顧者。本計劃以蘇格蘭國家腦退化症策略為本，採用蘇格蘭政府認可的「診斷後支援服務五大支柱」作藍本，為腦退化症人士及其家屬提供優質而適切的介入。並以家庭介入（Family Intervention）及充權（Empowerment）的模式，提倡「及早檢測、及早確診、及早介入」，協助上述人士制定合適的照顧計劃；而照顧者以嶄新角度了解其照顧的角色，增加正面照顧的經驗、知識和技巧，以減低照顧帶來的影響，與腦退化症人士同行。 |

| 2016-2018 | 賽馬會「腦動耆跡」計劃 | 腦退化症人士會因為認知能力退化，在處理個人事務及安排個人生活上逐漸出現困難，而隨着本港人口老化問題日益嚴重，如何建立一個腦退化症人士的友善社區成為一個重要的議題。 作為全方位推動腦退化症服務的機構，賽馬會耆智園認為協助建立一個腦退化症人士的友善社區是責無旁貸的工作。承蒙 香港賽馬會慈善信託基金捐助，本園於2014年8月至2015年10月期間舉辦了「腦動全城」公眾教育計劃，鼓勵社會上不同崗位的人士認識腦退化症，並在職場和社區中為有需要的腦退化症人士提供及時和適當的支援。有賴各方的積極參與，「腦動全城」計劃為公共及私營機構的超過4,000名前線人員提供了互動講座，並招募了逾180位「腦動全城」大使，一起把關愛腦退化症人士的訊息傳揚至社會各階層，總計共超過24,000人受惠，成績斐然。 為加強社區的支援力量，本園萬分感謝香港賽馬會慈善信託基金捐助，繼續支持本園於2016年10月至 2018年9月將舉行「賽馬會『腦動耆跡』計劃」，發掘更多願意為腦退化症人士提供協助的社區人士，使腦退化症人士於社區的各個角落也可以得到幫助，享受更愉快的生活。 |
