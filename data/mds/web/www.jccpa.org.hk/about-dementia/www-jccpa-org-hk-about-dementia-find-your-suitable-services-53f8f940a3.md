# 尋找合適服務 - 賽馬會耆智園 | 一站式腦退化症服務中心

###### 如懷疑自己或家人患有腦退化症或出現認知衰退的情況，應及早求醫及治療，此舉有助延緩衰退。而照顧腦退化症人士是一項富挑戰性的工作，照顧者除需要對此病有充份認識外，更應善用社區資源，以紓緩照顧壓力。以下的資訊有助尋找適當的服務及支援，亦可下載「社區資源」單張作參考。

[ 下載

](https://www.jccpa.org.hk/wp-content/uploads/2021/07/3.pdf)

## 診斷及評估途徑

如果家人的記性、性格、習慣、行為出現明顯的轉變，如經常遺失個人物品、表達及理解有困難、對時間及人物混亂、失去主動性、突然變得冷漠等，他們有可能患上腦退化症，應盡速尋求醫生的診斷，可聯絡：

1. 私人執業醫生
2. 賽馬會耆智園的記憶診所
3. 其他非政府機構的記憶診所
4. 部份政府醫院設有記憶診所／老人科／老齡精神科診斷服務。應診者必須具備醫生轉介信。

[ 耆智園的記憶診所詳情

](https://www.jccpa.org.hk/services/memory-clinic/)

[ 尋找其他記憶診所資料

](https://www.jccpa.org.hk/about-dementia/community-resources/)

## 院舍照顧服務

院舍照顧服務 當腦退化症人士身體情況變弱或照顧者未能繼續擔當照顧工作的時候，院舍照顧服務便是其中一個出路。院舍主要分為安老院、護理安老院、護養院及療養院。申請人可向區內家庭服務中心、醫務社會工作部、長者地區／鄰舍中心登記申請。

[ 尋找相關資料

](https://www.jccpa.org.hk/about-dementia/community-resources/)

短期住宿服務為了讓照顧者得到較長的休息時間或需暫停照顧工作(如往外地旅遊、入院、傭工放假等)，耆智園及部份院舍設有短暫住宿服務。

[ 耆智園的短期住宿服務詳情劃

](https://www.jccpa.org.hk/services/residential-service/)

[ 尋找其他相關資料

](https://www.jccpa.org.hk/about-dementia/community-resources/)

## 日間護理服務

腦退化症日間護理中心主要為腦退化症人士提供不同的活動及訓練，增加其自信心，維護他們的尊嚴、盡量讓他們發揮現有功能和減慢認知能力衰退的速度。除耆智園外，社會福利署、部份安老服務單位或私營護理中心，亦有為腦退化症長者提供日間護理及日間暫託服務。

[ 耆智園日間護理服務詳情

](https://www.jccpa.org.hk/services/day-care-service/)

[ 尋找其他相關資料

](https://www.jccpa.org.hk/about-dementia/community-resources/)

## 假日日間護理服務

為了滿足照顧者的需要，本中心設有星期日及假日日間護理服務，護理人員會安排充實的活動予參加者，務求令他們度過一個愉快的假日。

[ 耆智園的假日日間護理服務詳情

](https://www.jccpa.org.hk/services/day-care-service/)

## 照顧者支援小組

照顧者支援小組是一個互助小組，參與者大多為照顧者。透過定期聚會，讓家屬得到同伴及專業人士的支援。社工更為腦退化症人士家屬提供適合的輔導及協助。

[ 尋找相關資料

](https://www.jccpa.org.hk/about-dementia/community-resources/)

## 經濟援助
