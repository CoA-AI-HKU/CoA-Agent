# 溝通方式 - 賽馬會耆智園 | 一站式腦退化症服務中心

###### 也許你會覺得很難與腦退化症人士溝通，其實，造成溝通困難的原因很多，腦退化症人士的情況變化只是其中一個因素。溝通包括接收、理解和回應三個部份。照顧者可以改善自己的溝通方式，靈活使用不同方式去溝通，也可以協助腦退化症人士接收、理解和回應來改善溝通。只要以理解、寬容和體諒的態度面對腦退化症人士，便能打破溝通的障礙。

## 溝通困難的原因

### 照顧者

- 壓力
- 過高的期望
- 說話技巧

### 環境

- 在場人物
- 光線
- 聲音
- 氣味
- 溫度、濕度

### 腦退化症人士

- 五感退化
- 認知能力衰退
- 情緒困擾

## 溝通障礙

## 初期腦退化症人士
| | 原因 | 應對
| 接收 |

- 看得不清楚、聽得不清楚
- 注意力下降

- 為腦退化症人士準備適合的眼鏡和助聽器
- 避免突然轉換話題
- 需要時要歸納或重複內容
- 保持環境安靜、舒適
| 理解 |

- 認知能力退化，例如：未能理解太長的句子、未能理解太抽象的內容
- 專注力下降

- 用簡單、直接的語句
- 給予對方充足時間回應
| 回應 |

- 表達能力下降，例如：模稜兩可地回答以避免作解釋，如「係咁架啦」、「你話點就點啦」、「唔清楚」等
- 遣詞用字有困難，例如：以用空泛的語句代替人物╱物件╱地方，如「嗰度」、「嗰啲」、「嗰個人」等

- 鼓勵長者表達自己，例如：自己的情緒、對事物的感覺等
- 保持態度溫和
- 避免與長者爭論

## 中期腦退化症人士
| | 原因 | 應對
| 接收 |

- 接收速度緩慢，例如：需較長時間接收訊息
- 注意力進一步下降，例如：突然想起陳年往事，便不會留意別人說甚麼

- 溝通時面對腦退化症人士
- 對話時先稱呼對方及介紹自己的名字
- 保持適當的音量
- 留意對方的表情，如對方顯得疑惑不安，便使用更簡單的說法再說一次
- 保持環境安靜、舒適
| 理解 |

- 認知能力進一步退化，例如：只能理解簡單指示、只能理解眼前事物、只能理解母語
- 專注力進一步下降，例如：只回應句子內的一個詞語，造成答非所問

- 溝通時善用實物、語調、圖畫、文字、表情、手勢等，令對方更易掌握
- 給予對方更多時間理解及反應
- 避免用代名詞，如：他、他們等等
| 回應 |

- 表達能力進一步下降，例如：未能說出完整句子
- 遣詞用字有困難，例如：說錯人物、地點的名稱
- 失去聲音及語調運用的能力

- 保持態度溫和
- 多點提示和讚賞
- 避免與對方爭論

## 晚期腦退化症人士
| | 原因 | 應對
| 接收 |

- 接收能力有限，例如：不知道別人在和他說話

- 確定對方看到你後再進行溝通
- 對話時先稱呼對方及介紹自己的名字
- 保持適當的音量
- 多些笑容，並適當地給予身體接觸，例如：輕按手臂、手背等
- 保持環境安靜、舒適
| 理解 |

- 認知能力有限，例如：不能理解說話，只能理解動作及觸摸

- 溝通時善用實物、語調、圖畫、文字、表情、手勢等，令對方更易掌握
- 放慢說話速度，確保咬字清楚
| 回應 |

- 表達能力有限，例如：只能回答 「好」、「唔好」或難以理解的聲音
- 只能以動作、聲音等非語言方法來表達自己

- 即使對方缺乏反應，也要永遠相信他明白你的說話，持續溝通
- 多留意對方的非語言訊息
- 嘗試回應對方的不明反應，例如：對方握你的手時輕撫他的手背
