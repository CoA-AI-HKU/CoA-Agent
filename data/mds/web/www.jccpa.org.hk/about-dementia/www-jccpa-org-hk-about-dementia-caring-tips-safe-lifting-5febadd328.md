# 扶抱技巧 - 賽馬會耆智園 | 一站式腦退化症服務中心

## 坐直身

扶抱腦退化症人士，協助他 / 她從仰臥到坐起

1. 提高腦退化症人士雙膝

2. 把手放在腦退化症人士肩膀及盆骨，慢慢把他/她轉身至側臥，並把他/她的雙腿移至床邊，期間保持背部挺直

3. 把腦退化症人士的小腿放下床沿

4. 把前臂穿過腦退化症人士後頸，扶好他/她的頸部及肩膀，另一手扶著他/她的大腿，雙膝微曲，稍稍坐下，示意他從一數到三後坐直，數到三時用力扶起他/她的肩膀

5. 扶著腦退化症人士直至其坐起來，期間保持腰背挺直

## 單人扶持雙膝支撐法

扶抱腦退化症人士，協助他 / 她坐到其他地方

1. 確保已鎖上輪椅鎖掣

2. 站到腦退化症人士面前，確保他/她的雙腳平放，把其中一隻腳放到他/她的雙腿之間，站穩

3. 請腦退化症人士把雙手繞到你的後頸並扶好後，雙膝微曲，稍稍坐下，抱住他/她的髖關節部位

4. 示意腦退化症人士從一數到三後抓緊你並站起，數到三時用雙腿的力量提舉及慢慢移動他/她，期間保持腰背挺直

5. 移動腦退化症人士，讓他/她轉移到輪椅前，再扶他/她慢慢坐到輪椅上，期間保持腰背挺直

## 穿臂握法

提舉腦退化症人士，協助他 / 她坐好

1. 確保已鎖上輪椅鎖掣

2. 請腦退化症人士交叉雙手後，在他/她背後站穩

3. 前臂穿過腦退化症人士腋下，右手握住他/她的左腕，左手握住他/她的右腕

4. 雙膝微曲，稍稍坐下，示意腦退化症人士從一數到三後要扶他坐直，數到三時用全身力量提舉他/她，期間保持腰背挺直

5. 確保腦退化症人士坐回坐墊上才放手
